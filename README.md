> A clean and intuitive code snippet manager that helps developers efficiently store, organize, and reuse code.

### Features Include

- Syntax highlighting (powered by <a href="https://shiki.style" target="_blank">Shiki</a>)
- Tag-based classification
- Fast local persistence (IndexedDB)
